# スマートビュー

サイドバー上部のビュー切り替えで、ブックマークの表示をコンテキストに応じて変更できます。

## ビューの種類

### すべて
登録されているすべてのブックマークを表示します（デフォルト）。

### 未分類
タグが一つも設定されていないブックマークのみを表示します。AI アシスタントで一括タグ付けするのに便利です。

### 最近追加
過去 7 日以内に追加されたブックマークを表示します。

### 使われていない
`visit_count = 0` のブックマークを表示します。ブックマークの整理に活用できます。

### よく見る
訪問回数が多いブックマークを降順に表示します。

## デバイススコープ

サイドバー下部のスコープ切替で、表示範囲を制御できます。

| スコープ | 表示内容 |
|---|---|
| すべて | 全ブックマーク |
| 共有のみ | `device_scoped = false` のブックマークのみ |
| デバイス固有のみ | `device_scoped = true` のブックマークのみ |

スコープはスマートビューと組み合わせて使えます。たとえば「デバイス固有のみ」＋「最近追加」で、今使っているデバイスで最近追加したブックマークだけを見ることができます。

## タグフィルタ

サイドバーのタグ一覧からタグをクリックすると、そのタグが付いたブックマークのみに絞り込みができます。スマートビュー・スコープと同時に適用可能です。
